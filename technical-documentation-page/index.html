Introduction

HTML stands for HyperText Markup Language and is the standard language used to create web pages.

HTML provides the structure of a webpage using elements such as headings, paragraphs, images, and links.

<h1>Hello, World!</h1>
HTML Elements

HTML elements are the building blocks of every webpage.

Most elements use an opening tag, content, and a closing tag.

  • Headings
  • Paragraphs
  • Images
<p>This is a paragraph.</p>
Attributes

Attributes provide additional information about an HTML element.

They are written inside the opening tag.

  • href
  • src
  • alt
<a href="https://example.com">Visit Website</a>
Lists

HTML supports ordered and unordered lists.

Lists help organize related information.

  • Unordered List
  • Ordered List
            
                <ul>
                    <li>Apple</li>
                    <li>Banana</li>
                </ul>
Forms

Forms allow users to enter and submit information.

Common form elements include inputs, labels, and buttons.

  • Text Input
  • Checkbox
  • Submit Button
            
                <form>
                    <input type="text">
                    <button>Submit</button>
                </form>
